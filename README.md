# Data Analysis Dashboard (Power BI)

## О проекте
Аналитический дашборд, созданный в Power BI для анализа демографических показателей стран.

## Данные
- country — страна  
- region — регион  
- median_age — медианный возраст  
- birth_rate — рождаемость  
- death_rate — смертность  
- life_expectancy — продолжительность жизни  
- fertility_rate — фертильность  
- obesity_rate — уровень ожирения  

## Что анализировалось
- сравнение стран
- влияние демографии на продолжительность жизни
- распределение показателей

## Вывод
- развитые страны имеют выше продолжительность жизни  
- различия между регионами значительные  
- демография сильно влияет на качество жизни  

## Технологии
- Power BI
